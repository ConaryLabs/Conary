# R2 + Local Chunk Garbage Collection Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Delete orphaned chunks from R2 and local disk that no converted package references anymore.

**Architecture:** Single-pass GC reads the referenced set from `converted_packages.chunk_hashes_json`, scans local disk + R2 for stored chunks, deletes the difference. Exposed via CLI flag and MCP tool. Dry-run mode for safety.

**Tech Stack:** Rust 1.94, rusqlite, rust-s3 (R2Store), tokio.

**Spec:** `docs/superpowers/specs/2026-03-16-r2-chunk-gc-design.md`

```rust
/// List all chunk hashes stored in R2.
/// Paginates through all objects with the configured prefix.
pub async fn list_chunks(&self) -> Result<Vec<String>> {
    let mut all_hashes = Vec::new();
    let mut continuation_token: Option<String> = None;

    loop {
        let results = self.bucket
            .list(self.prefix.clone(), None)
            .await
            .context("Failed to list R2 objects")?;

        for item in &results {
            for object in &item.contents {
                if let Some(hash) = object.key.strip_prefix(&self.prefix) {
                    if !hash.is_empty() {
                        all_hashes.push(hash.to_string());
                    }
                }
            }

            continuation_token = item.next_continuation_token.clone();
        }

        if continuation_token.is_none() {
            break;
        }
    }

    Ok(all_hashes)
}
```

NOTE: The exact S3 list API may differ based on the `s3` crate version.
Read the existing `r2.rs` code to match the crate's API. The `s3::Bucket`
may use `list()`, `list_page()`, or `list_objects_v2()`. Adapt accordingly.

```rust
/// Collect all chunk hashes referenced by converted packages
/// and protected chunks.
pub fn build_referenced_set(conn: &Connection) -> Result<HashSet<String>> {
    let mut referenced = HashSet::new();

    // From converted_packages.chunk_hashes_json
    let mut stmt = conn.prepare(
        "SELECT chunk_hashes_json FROM converted_packages
         WHERE chunk_hashes_json IS NOT NULL"
    )?;
    let mut rows = stmt.query([])?;
    while let Some(row) = rows.next()? {
        let json_str: String = row.get(0)?;
        if let Ok(hashes) = serde_json::from_str::<Vec<String>>(&json_str) {
            for hash in hashes {
                referenced.insert(hash);
            }
        }
    }

    // Protected chunks from chunk_access
    let mut stmt = conn.prepare(
        "SELECT hash FROM chunk_access WHERE protected = 1"
    )?;
    let mut rows = stmt.query([])?;
    while let Some(row) = rows.next()? {
        let hash: String = row.get(0)?;
        referenced.insert(hash);
    }

    Ok(referenced)
}
```

- [ ] **Step 3: Implement scan_local_chunks**

Use the existing `scan_chunk_hashes()` from `handlers/chunks.rs` or
reimplement inline. The local chunk directory uses a two-level structure:
`{objects_dir}/{hash[0:2]}/{hash[2:]}`.

```rust
/// Scan local disk for all stored chunk hashes.
pub fn scan_local_chunks(objects_dir: &Path) -> Result<Vec<String>> {
    let mut hashes = Vec::new();
    if !objects_dir.exists() {
        return Ok(hashes);
    }

    for prefix_entry in std::fs::read_dir(objects_dir)? {
        let prefix_entry = prefix_entry?;
        if !prefix_entry.file_type()?.is_dir() {
            continue;
        }
        let prefix = prefix_entry.file_name().to_string_lossy().to_string();
        if prefix.len() != 2 {
            continue;
        }
        for file_entry in std::fs::read_dir(prefix_entry.path())? {
            let file_entry = file_entry?;
            let suffix = file_entry.file_name().to_string_lossy().to_string();
            if suffix.ends_with(".tmp") {
                continue;
            }
            hashes.push(format!("{}{}", prefix, suffix));
        }
    }

    Ok(hashes)
}
```

- [ ] **Step 4: Implement the main GC function**

```rust
pub struct GcResult {
    pub local_scanned: usize,
    pub r2_scanned: usize,
    pub referenced: usize,
    pub protected: usize,
    pub local_deleted: usize,
    pub r2_deleted: usize,
    pub local_bytes_freed: u64,
    pub r2_bytes_freed: u64,
}

/// Run chunk garbage collection.
///
/// `dry_run`: if true, report what would be deleted without deleting.
/// `r2_store`: if Some, also GC the R2 bucket.
/// `grace_period_secs`: skip chunks accessed within this window (default 3600).
pub async fn run_chunk_gc(
    db_path: &Path,
    objects_dir: &Path,
    r2_store: Option<&crate::server::r2::R2Store>,
    dry_run: bool,
    grace_period_secs: u64,
) -> Result<GcResult> {
    let conn = conary_core::db::open(db_path)?;

    // Step 1: Build referenced set
    let referenced = build_referenced_set(&conn)?;
    info!("Referenced: {} chunks", referenced.len());

    // Step 2: Scan local
    let local_chunks = scan_local_chunks(objects_dir)?;
    info!("Local: {} chunks on disk", local_chunks.len());

    // Step 3: Scan R2
    let r2_chunks = if let Some(store) = r2_store {
        store.list_chunks().await.unwrap_or_default()
    } else {
        Vec::new()
    };
    info!("R2: {} chunks in bucket", r2_chunks.len());

    // Step 4: Find local orphans (with grace period)
    let cutoff = std::time::SystemTime::now()
        - std::time::Duration::from_secs(grace_period_secs);
    let mut local_deleted = 0usize;
    let mut local_bytes = 0u64;

    for hash in &local_chunks {
        if referenced.contains(hash) {
            continue;
        }
        let path = objects_dir
            .join(&hash[..2])
            .join(&hash[2..]);

        // Grace period: skip recently accessed files
        if let Ok(meta) = std::fs::metadata(&path) {
            if let Ok(accessed) = meta.accessed() {
                if accessed > cutoff {
                    continue;
                }
            }
            local_bytes += meta.len();
        }

        if dry_run {
            info!("[dry-run] Would delete local: {}", hash);
        } else {
            if let Err(e) = std::fs::remove_file(&path) {
                tracing::warn!("Failed to delete local chunk {}: {}", hash, e);
            } else {
                info!("Deleted local: {} ({} bytes)", hash, local_bytes);
                // Clean up empty parent directory
                let parent = objects_dir.join(&hash[..2]);
                let _ = std::fs::remove_dir(&parent); // ignore if not empty
            }
        }
        local_deleted += 1;
    }

    // Step 5: Find R2 orphans
    let mut r2_deleted = 0usize;
    let mut r2_bytes = 0u64;

    if let Some(store) = r2_store {
        for hash in &r2_chunks {
            if referenced.contains(hash) {
                continue;
            }
            if dry_run {
                info!("[dry-run] Would delete R2: {}", hash);
            } else {
                if store.delete_chunk(hash).await {
                    info!("Deleted R2: {}", hash);
                } else {
                    tracing::warn!("Failed to delete R2 chunk: {}", hash);
                }
            }
            r2_deleted += 1;
        }
    }

    // Step 6: Clean up chunk_access for deleted local chunks
    if !dry_run && local_deleted > 0 {
        // Remove orphaned chunk_access entries
        for hash in &local_chunks {
            if !referenced.contains(hash) {
                let _ = conn.execute(
                    "DELETE FROM chunk_access WHERE hash = ?1",
                    rusqlite::params![hash],
                );
            }
        }
    }

    Ok(GcResult {
        local_scanned: local_chunks.len(),
        r2_scanned: r2_chunks.len(),
        referenced: referenced.len(),
        protected: 0, // counted during build_referenced_set
        local_deleted,
        r2_deleted,
        local_bytes_freed: local_bytes,
        r2_bytes_freed: r2_bytes,
    })
}
```

```rust
#[derive(Debug, Deserialize, JsonSchema)]
struct ChunkGcParams {
    #[schemars(description = "Show what would be deleted without deleting")]
    dry_run: Option<bool>,
}

#[tool(description = "Garbage collect orphaned chunks from local disk and R2. Finds chunks not referenced by any converted package and deletes them.")]
async fn chunk_gc(&self, Parameters(params): Parameters<ChunkGcParams>) -> Result<CallToolResult, McpError> {
    let state = self.state.read().await;
    let db_path = state.config.db_path.clone();
    let objects_dir = state.config.chunk_dir.clone();
    let r2_store = state.r2_store.clone(); // Option<Arc<R2Store>>
    drop(state);

    let dry_run = params.dry_run.unwrap_or(false);

    let result = crate::server::chunk_gc::run_chunk_gc(
        &db_path,
        &objects_dir,
        r2_store.as_deref(),
        dry_run,
        3600,
    )
    .await
    .map_err(service_err_to_mcp)?;

    to_json_text(&serde_json::json!({
        "dry_run": dry_run,
        "local_scanned": result.local_scanned,
        "r2_scanned": result.r2_scanned,
        "referenced": result.referenced,
        "local_deleted": result.local_deleted,
        "r2_deleted": result.r2_deleted,
        "local_bytes_freed": result.local_bytes_freed,
        "r2_bytes_freed": result.r2_bytes_freed,
    }))
}
```
